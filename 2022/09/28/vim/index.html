    vim简介

    vim一般有三种模式

    • 普通模式

      NORMAL MODE 使用ESC按键可以从编辑模式退回到普通模式

    • 编辑模式

      INSERT MODE 进入编辑模式就可以对文件进行正常的编辑，从普通模式进入编辑模式常用有以下几种方式

      • i 输入i从光标位置进入编辑模式，insert
      • a 输入a从光标的下一个位置进入编辑模式，append
      • o 输入o从光标的下一行新创建一行进入编辑模式，open a line below
      • I 输入I从光标行的行首位置进入编辑模式，insert before line
      • A 输入A从光标行的行尾位置进入编辑模式，append after line
      • O 输入O从光标的上一行新创建一行进入编辑模式，open a line above

    • 命令模式

      COMMAND MODE 普通模式下输入:之后执行命令，例如：

      • :w 保存
      • :q 退出vim
      • :vs or :sp 分屏 vs(vertical split) sp(split)
      • :% s/{str}/{str}/g 全局替换

    • 可视化模式

      VISUAL MODE 一般用来块状选择文本

      • v 输入v 进入VISUAL选择
      • V 输入V 进入VISUAL LINE选择行
      • Ctrl + V 输入Ctrl + v 进入VISUAL BLOCK选择方块选择

    编辑模式

    常用的一些快捷键

    • Ctrl + h 删除上一个字符
    • Ctrl + w 删除上一个单词
    • Ctrl + u 删除当前行
